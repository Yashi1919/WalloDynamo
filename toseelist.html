Welcome to ToSeeList

ToSeeList is a simple React Native application that allows users to manage wallpapers with custom timings. Below are the key features and instructions on how to use the application.

Features

  • Add frames (wallpapers) by uploading images from your device.
  • Select a specific time for each wallpaper to be set as your device's wallpaper.
  • Toggle the wallpaper to enable or disable the scheduled wallpaper change.
  • View a list of added wallpapers along with their scheduled times.
  • Delete wallpapers you no longer want to manage.

How to Use ToSeeList

1. Adding a Wallpaper Frame

To add a new wallpaper:

  • Tap the + button located at the bottom center of the screen.
  • A photo selection interface will appear; choose a photo from your library.
  • The selected photo will be added to your frames list.

2. Setting a Time for Your Wallpaper

To set a specific time for your wallpaper:

  • Find the wallpaper frame you want to configure in the list.
  • Tap the "Select Time" button to open the time picker.
  • Choose the desired time and confirm your selection.

3. Toggling the Wallpaper

To enable or disable a wallpaper:

  • Locate the frame in the list and use the toggle switch next to it.
  • If the time is not set, you will be prompted to set the time before enabling the toggle.

4. Deleting a Wallpaper Frame

To delete a wallpaper:

  • Long press on the image of the frame you want to delete.
  • A confirmation dialog will appear; select "OK" to delete the frame.

5. Viewing Your Frames

If there are no frames added, a message will be displayed indicating that you can click the + button to add new tasks.

Important Notes

Note: Make sure that the application has the necessary permissions to access your photo library to successfully upload images.
