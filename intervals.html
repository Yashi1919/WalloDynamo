Intervals

Overview

The RandomWallpaper component allows users to change their device wallpaper at specified intervals using images from their library. Users can add images, set a time interval, and toggle the wallpaper change on and off.

Features

  • Add images from the device's image library.
  • Set time intervals for wallpaper changes (in seconds, minutes, or hours).
  • Toggle wallpaper changes on or off.
  • View and remove added images from the list.

How to Use

1. Adding Images

To add images from your library, click the "Add Images from Library" button. You can select multiple images, and they will be displayed in a grid format.

2. Setting Time Intervals

Enter a numeric value in the input field (e.g., 5) to specify the time interval. Use the dropdown to select the unit of time (seconds, minutes, or hours) for the wallpaper change.

3. Toggling Wallpaper Change

Use the toggle switch labeled "Enable Wallpaper Change" to start or stop the automatic wallpaper changes. If the toggle is on, the wallpaper will change at the intervals you set.

4. Removing Images

To remove an image, long press on the image in the list. You will be prompted to confirm the deletion.

Notes

Important: Make sure to allow necessary permissions for accessing the device's photo library and enabling background tasks for seamless wallpaper changes.

Troubleshooting

If you encounter any issues, ensure that you have the required libraries installed and that you are testing the component on a compatible device.
