Instructions for Using the Birthday Wallpaper App

Getting Started

  • Download and install the app on your device.
  • Open the app and allow any necessary permissions.

Adding Birthdays

  • Click the "+" button at the bottom of the screen to add a new birthday.
  • Select a photo for the birthday by tapping on "Upload Photo."
  • Choose a date for the birthday by tapping on "Select Date."
  • The birthday will be saved automatically.

Managing Birthdays

  • To change a birthday photo, tap on the existing photo.
  • To toggle the wallpaper setting for a birthday, use the switch next to the birthday entry.
  • Long press on a birthday entry to delete it.

Setting Wallpapers

  • Once a birthday date is set and the toggle is enabled, the wallpaper will automatically be applied on the specified date.
  • Ensure the date is in the future; otherwise, the wallpaper will be set for the following year.

Notes

Please remember to keep the app running in the background for wallpaper changes to occur at the scheduled time.

Contact

If you encounter any issues, please contact support@example.com for assistance.
